Chanel Mepschen is een frontend developer bij Tripel in Alkmaar. Zelf is ze ook als CMDÃ©r afgestudeerd. Ze heeft het over hoe je een frontend setup in productie zet. Tripel is een groot technisch bedrijf met veel verschillende projecten. Ook grote klanten zoals Ajax, Heiniken, NLZiet en Max Verstappen.

Bij Tripel werken ze met veel frameworks want op zo'n manier kan je makkelijk stukke code hergebruiken. Dit zijn dan herbruikbare componenten. Dit zorgt ervoor dat je als developer je zelf niet steeds herhaalt. Ook is het makkelijk om zo'n component te updaten. Je hoeft het maar op 1 plek te doen en het wordt overal geupdate. Ze werkte eerst veel met React maar omdat bij react content niet meteen geladen wordt. Komen applicaties/websites niet hoog in de zoekresultaten van Google. Daarom zijn ze overgestapt naar Next.js. Dit is een framework dat gebouwd is op React. Het zorgt ervoor dat de content wel meteen geladen wordt. Dit zorgt ervoor dat de websites/applicaties wel hoog in de zoekresultaten van Google komen

Naast NextJS gebruiken ze ook typescript. Dit is een superset van javascript. Het is een manier om javascript te schrijven met types. Dit zorgt ervoor dat je minder fouten maakt. Ook is het makkelijker om te lezen.

Om de zelfde structuur aan te houden met coderen binnen het bedrijf gebrbuikt Tripel CSS formatters. Hierbij kan je denken aan Stylelint, Prettier en Eslint. Dit zorgt ervoor dat iedereen binnen het bedrijf dezelfde code schrijft. Ook zorgt het ervoor dat je minder fouten maakt. Dit komt omdat je niet meer zelf hoeft te bedenken hoe je de code schrijft. Dit wordt voor je gedaan.
