Sophie Koonin

Sophie Koonin is een front-end developer bij het Financial Times-team in Londen. Ze is een van de oprichters van de London Web Performance Group, een groep die regelmatig bijeenkomt om te praten over webprestaties. Zij vindt dat het web nu best wel saai is geworden en dat iedereen hetzelfde doet. Ze wil dat mensen weer creatief worden en dat ze weer gaan experimenteren. Ze wil dat mensen weer gaan spelen met het web. MAKE CSS WEIRD AGAIN! Ze liet een aantal websites zien dei in de borige decinia gemaakt werden, zo zag je een wereld van verschil en hoe erg het web veranderd is. Ze liet ook een aantal websites zien die ze zelf gemaakt had. Ze liet zien hoe je met css animaties kan maken en hoe je met css kan spelen. Maar omdat het vroeger een beetje uit de hand liep met de css animaties en de gifjes kunnen wij nu voor de gebruikers die het niet willen de animaties uitzetten met reduce motion.

Adam Argyle

Adam Argyle is een ontwerper en ontwikkelaar die graag dingen maakt voor het web. Hij werkt bij het Chrome-team van Google, waar hij helpt bij het ontwerpen en bouwen van tools voor ontwikkelaars. Hij werkt ook bij de CSS Working Group, waar hij helpt bij het ontwerpen en bouwen van het webplatform. Hij woont in Seattle met zijn vrouw en twee kinderen. Tijdens pre-css-day gaf hij een korte presentatieover kleur en hoe je kleur kan gebruiken in je css. je zou denken dat kleur... ja dat kennen we wel. Maar Adam liet zien dat er veel meer is dan alleen de kleur die je ziet. Hij introduceerde olchk. Hiermee hoef je maar 1 kleur te grbuiken van de hue diagram en kan je gebaseerd of helderheid en verzadiging de rest van de kleuren bepalen. Dit is een hele handige tool om te gebruiken als je een kleur thema wil maken voor je website. Hiermee kan je makkelijk een kleuren pallet maken en de contrasten checken.

Beoordeling

De pre-CSS-day krijgt van mij alle steren omdat ik het erg interessant vond en met tweee korte speaches veel heb geleerd. Dingen die ik zeker ga toepassen in mijn eigen projecten. Ik vond het erg leuk om te zien hoe je met css kan spelen en hoe je met kleur kan spelen. En ook hoe makkelijk je een kleuren pallet kan maken met olchk.
